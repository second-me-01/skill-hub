# 小红书爆款内容公式库

## 一、爆款标题公式（15 个）

### 1. 数字清单型
**公式**：`数字 + 核心关键词 + 利益点/人群`
- 7 个平价好物让你的出租屋焕然一新
- 5 款学生党必入的百元蓝牙耳机
- 30 天自律计划｜我的人生开挂了

### 2. 反常识/颠覆认知型
**公式**：`别再 XXX 了 + 正确做法/真相`
- 别再乱买精华了！皮肤科医生教你选
- 求你别再这样洗脸了❌正确方法在这
- 原来我们一直都用错了❗xxx 的正确打开方式

### 3. 痛点解决型
**公式**：`痛点描述 + 解决方案/效果`
- 毛孔粗大有救了😭这个方法真的绝
- 拖延症晚期自救指南｜亲测有效
- 英语口语差到不敢开口？试试这个方法

### 4. 身份共鸣型
**公式**：`身份标签 + 场景/痛点 + 方案`
- 打工人下班后的极简晚餐🍳15分钟搞定
- 小个子女生穿搭｜显高10cm不是梦
- 新手妈妈必看‼️这些坑我替你踩过了

### 5. 对比冲突型
**公式**：`A vs B / 前后对比 + 效果`
- 早C晚A一个月｜皮肤变化也太大了
- 100元 vs 1000元粉底液真实对比
- 改造前 vs 改造后｜我的5㎡小卧室逆袭

### 6. 好奇心驱动型
**公式**：`悬念/不完整信息 + 情绪词`
- 后悔没早点知道的厨房神器😭
- 这个小众品牌凭什么火遍全网？
- 被闺蜜安利的 xxx 到底值不值得买

### 7. 权威背书型
**公式**：`专业身份/数据 + 推荐/建议`
- 护肤成分党｜成分表看不懂的看这篇
- 健身教练私藏的 5 个燃脂动作🔥
- 美术老师教你 3 步画出高级感配色

### 8. 时间紧迫型
**公式**：`时间限定 + 效果承诺`
- 3天速成！零基础学会手机修图
- 每天10分钟｜一个月练出马甲线
- 周末两天就能完成的房间大改造

### 9. 省钱/平替型
**公式**：`价格优势 + 大牌对标 + 效果`
- 9.9元平替！效果不输大牌的 xxx
- 月薪3000也能拥有的精致生活
- 不花一分钱｜在家就能做的 spa

### 10. 情绪价值型
**公式**：`强情绪词 + 场景/产品`
- 幸福感爆棚的 10 件家居好物✨
- 真的会谢😭用了这个再也回不去了
- 一整个爱住‼️今年最满意的购物清单

### 11. 场景代入型
**公式**：`具体场景 + 解决方案`
- 约会前 2 小时急救妆容💄保姆级教程
- 出差行李箱怎么装？一个登机箱搞定 5 天
- 面试穿什么？HR 最爱的职场穿搭

### 12. 盘点合集型
**公式**：`时间/主题 + 合集/盘点`
- 2024 年度爱用物大盘点🏆真心推荐
- 去xx必吃的 10 家店｜本地人推荐
- 入住半年｜最值得买的家居好物合集

### 13. 挑战/实验型
**公式**：`挑战行为 + 时间 + 结果`
- 连续 30 天早起 5 点｜我的变化太大了
- 挑战一周只花 100 元吃饭🍚
- 坚持跳绳一个月的身体变化记录

### 14. 避坑/踩雷型
**公式**：`避坑/踩雷 + 领域 + 经验教训`
- 装修踩过的 8 个大坑⚠️千万别再犯
- 新手化妆避坑‼️这些错误我全犯过
- 租房避雷指南｜这 5 种房子千万别租

### 15. 教程/保姆级型
**公式**：`保姆级/手把手 + 教程主题`
- 保姆级教程🔥零基础学会做面包
- 手把手教你注册 xxx｜小白也能学会
- 超详细攻略‼️第一次去 xxx 看这篇就够

## 三、Emoji 使用指南

### 使用原则
- 标题中使用 1-2 个 emoji，放在**开头或关键词旁边**
- 正文中 emoji 主要用于**小标题/分隔符**，不要每句话都加
- 整篇笔记 emoji 数量控制在 **15-25 个**，过多显得杂乱
- 选择与内容调性匹配的 emoji，不要随意使用

### 高频 Emoji 速查表

**通用强调类**
- ‼️ 强调重点 / ⚠️ 注意提醒 / 💡 小技巧
- ✅ 推荐/正确 / ❌ 不推荐/错误
- 📌 要点标注 / 📍 地点标记
- ⭐ 评分/推荐 / 🔥 热门/火爆
- ✨ 亮点/好评 / 💯 满分推荐

**情绪表达类**
- 😍 超级喜欢 / 🥺 心动/感动 / 😭 感动到哭/后悔
- 🤔 思考/疑问 / 😱 惊讶/震惊 / 🥰 幸福/满足
- 💪 加油/坚持 / 🎉 庆祝/达成

**内容分类类**
- 🏠 家居 / 👗 穿搭 / 💄 美妆 / 🍳 美食
- 📚 学习 / 💻 数码 / ✈️ 旅行 / 🏋️ 健身
- 💰 省钱 / 🎁 好物 / 📸 拍照 / 🎨 设计

**序号替代**
- 1️⃣ 2️⃣ 3️⃣ 数字序号
- 🔴🟡🟢 重要程度标记
- 💎 核心/精华标记

### 不同笔记类型的 Emoji 风格

| 类型 | 风格 | 常用 emoji |
|------|------|------------|
| 种草文 | 热情、兴奋 | 😍✨🔥💯‼️ |

## 四、标签组合策略

### 「3+3+3」组合法则

#### 第一层：热门大标签（3 个）
搜索量 100w+，增加曝光基础盘。

常用万能标签：
- #好物推荐 #日常分享 #生活记录 #我的日常
- #学习打卡 #干货分享 #经验分享
- #穿搭分享 #美食分享 #家居好物

#### 第二层：精准中标签（3 个）
搜索量 10w-100w，精准触达目标人群。

按领域举例：
- 护肤：#护肤心得 #敏感肌护肤 #成分党
- 穿搭：#通勤穿搭 #小个子穿搭 #日系穿搭
- 美食：#家常菜 #减脂餐 #一人食
- 家居：#出租屋改造 #小户型收纳 #ins风家居
- 学习：#考研经验 #英语学习 #读书笔记

#### 第三层：长尾小标签（3 个）
搜索量 < 10w，竞争小容易获得排名。

构造方法：
- 品牌 + 产品名：#兰蔻小黑瓶 #戴森吹风机
- 场景 + 产品类型：#通勤包推荐 #夏日防晒霜
- 身份 + 需求：#大学生穿搭 #新手妈妈好物
- 地点 + 品类：#北京探店 #成都美食

### 标签放置位置

- 放在**正文末尾**，与正文空一行
- 也可以嵌入正文中的关键句后面（自然植入）
- 不要放在标题中（占标题字数且影响阅读体验）

### 标签注意事项

- 不要用与内容无关的热门标签（算法会判定为蹭流量，降权）
- 避免使用已被限流的标签（可在小红书搜索验证，如搜索结果异常少则可能被限）
- 每篇笔记标签不超过 10 个
- 品牌标签在非合作推广中谨慎使用

## 五、违禁词/敏感词清单

> 以下为常见易触发限流/违规的词汇，创作时应主动规避或替换。

### 1. 绝对化用语（广告法禁用）
| 违禁词 | 替代表达 |
|--------|----------|
| 最好/最佳 | 我用过里面很喜欢的 |
| 第一/No.1 | 很受欢迎的/口碑很好的 |
| 全网最低价 | 我买到的价格很划算 |
| 独家/唯一 | 比较少见的/很特别的 |
| 100% 有效 | 亲测体验不错 |
| 永久/永远 | 持续使用感觉不错 |
| 绝对/肯定 | 个人感觉/我觉得 |
| 秒杀/碾压/吊打 | 相比之下我更喜欢 |

### 2. 虚假宣传词
| 违禁词 | 替代表达 |
|--------|----------|
| 根治/治愈 | 有所改善/有帮助 |
| 特效/神药 | 效果还不错的 |
| 立竿见影 | 用了一段时间感觉到变化 |
| 包治/包好 | 可以试试 |
| 纯天然/零添加 | 成分比较温和 |
| 医学级/医用级 | [直接说产品名，不加修饰] |

### 3. 引流违规词
| 违禁词 | 说明 |
|--------|------|
| 加微信/加V | 严禁站外引流 |
| 私聊转账 | 涉嫌私下交易 |
| 淘宝搜索/拼多多搜 | 站外引流 |
| 点击链接/复制口令 | 站外引流 |
| 想要的扣 1 | 疑似引流话术 |
| 评论区留邮箱 | 站外引流 |

### 4. 诱导互动词（过度引导会被判定违规）
| 违禁词 | 替代表达 |
|--------|----------|
| 求求点赞收藏 | 觉得有用可以收藏备用 |
| 不转不是中国人 | [删除，不要用] |
| 双击屏幕有惊喜 | [删除，不要用] |
| 关注我必回关 | 欢迎关注一起交流 |

### 5. 其他敏感类型
- **政治敏感**：涉及国家领导人、政治事件、军事等话题
- **医疗健康**：未经证实的疗效宣传，处方药推荐
- **金融理财**：收益承诺、理财产品推荐（无资质）
- **未成年相关**：涉及未成年人的商业推广
- **版权相关**：未授权使用他人图片、文字、音乐

### 安全表达公式

当需要表达正面评价时，使用以下句式：
- "我个人觉得 xxx 还蛮不错的"
- "亲测用了 [时间]，体验分享给大家"
- "对我来说效果挺明显的（个人体质仅供参考）"
- "在我用过的 [品类] 里算是很满意的了"
- "不一定适合所有人，但我觉得值得一试"

这些表达既传递了正面态度，又避免了绝对化用语和虚假宣传的风险。
