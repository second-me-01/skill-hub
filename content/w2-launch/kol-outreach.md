# KOL 首批体验邀请

> 策略：3-5 个 AI 工具博主，给"首批推荐官"身份

## 目标 KOL 画像

- AI 工具测评博主（YouTube/B站/小红书）
- Claude Code 深度用户
- 粉丝量 1K-50K（腰部博主，合作门槛低、转化率高）
- 有技术背景，能真实体验产品

```

npx skills add secondme-team/skill-hub --all -g

包括：头脑风暴、设计系统生成、小红书文案、视频脚本、飞书集成等。

想邀请你作为首批推荐官体验一下。我们可以提供：
- 产品优先体验 + 直接技术支持

如果你感兴趣，我可以先给你发一份详细的功能介绍？

谢谢！
```

## 跟进节奏

| 天数 | 动作 |
|------|------|
| Day 0 | 发送邀请 DM |
| Day 2 | 如未回复，换一个平台再发一次 |
| Day 5 | 如已回复感兴趣，发送详细功能文档 + 安装指南 |
| Day 7 | 如已体验，请求产出内容（测评视频/帖子） |
| Day 10 | 内容发布，在各平台互动转发扩大影响 |
