# 视频脚本模板库 / Video Script Templates

本文档提供 5 种主流视频类型的完整脚本模板、Hook 技巧、分镜表格模板和字幕规范，供 video-script Skill 在创作时引用。

---

## 一、前3秒黄金法则 (The 3-Second Hook Rule)

短视频的前3秒决定了 80% 的完播率。以下是经过验证的高效 Hook 技巧：

### 1. 痛点提问型 (Pain Point Question)
> "你是不是也遇到过 XXX？"
> "为什么你的视频总是没人看？"
> "Are you still struggling with XXX?"

**适用场景：** 教程、口播、带货
**原理：** 让观众产生"说的就是我"的代入感

### 2. 反常识声明型 (Counter-Intuitive Statement)
> "90%的人不知道，早起其实是低效的"
> "千万别再用这个功能了，会毁掉你的手机"
> "Everything you know about XXX is wrong"

**适用场景：** 知识分享、口播
**原理：** 制造认知冲突，激发好奇心

### 3. 数据冲击型 (Data Shock)
> "我用这个方法，3天涨了10万粉"
> "月薪3千到3万，我只做了一件事"
> "I made $10K in 7 days using this method"

**适用场景：** 带货、经验分享、教程
**原理：** 具体数字增加可信度和吸引力

### 4. 悬念制造型 (Suspense Builder)
> "看到最后，你会感谢我"
> "接下来发生的事，让所有人都惊了"
> "Watch till the end — you won't believe what happens"

**适用场景：** 情景剧、Vlog、故事类
**原理：** 利用信息缺口驱动观看

### 5. 视觉冲击型 (Visual Impact)
> 直接展示最震撼的结果画面（before/after 对比）
> 展示一个出人意料的画面，再回到开头讲解

**适用场景：** 教程（先展示成果）、变装、美食
**原理：** 画面直接传递信息，比语言更快

### 6. 情绪共鸣型 (Emotional Resonance)
> "说真的，我真的受够了 XXX"
> "每次遇到这种情况，我都想..."
> "I'm so tired of XXX, and I know you are too"

**适用场景：** 口播、情景剧、吐槽类
**原理：** 情绪传染，快速建立共鸣

### Hook 组合公式

最强的 Hook 是组合使用，例如：
- **痛点 + 数据**："你还在熬夜加班吗？我用这个工具每天省了3小时"
- **反常识 + 悬念**："你以为喝8杯水是健康的？看完这个视频你就不这么想了"
- **视觉 + 数据**：[展示惊艳成果] + "只需要3步，0成本"

## 二、5种视频类型完整模板

### 模板 1：口播稿 (Talking Head / 知识分享)

**适用平台：** 抖音、B站、YouTube、TikTok
**推荐时长：** 60秒（短）/ 3-5分钟（中）/ 8-15分钟（长）

```
【标题】[数字] + [反常识/痛点] + [结果承诺]
示例：3个被你忽略的手机功能，最后一个能省一半电

【开头 Hook（0-3秒）】
[痛点提问/反常识声明]
示例："你手机里有个功能，可以帮你省50%的电，但99%的人都不知道"

【过渡（3-5秒）】
"今天我就来告诉你这[数字]个隐藏功能"
[停顿0.5s]

【正文-要点1（5-20秒）】
"**第一个**，[功能名称]。"
"你打开设置，找到 XXX，[具体操作步骤]"
"这个功能的好处是 [效果]"

【正文-要点2（20-35秒）】
"**第二个**，更厉害。"
"[具体内容]"
"我自己用了之后，[亲身体验]"

【正文-要点3（35-50秒）】
"**最后一个**，也是最重要的。"
"[具体内容]"
"[强调效果]"

【总结 + CTA（50-60秒）】
"好了，这3个功能你记住了吗？"
"觉得有用的话，**点个关注**，我每天分享一个实用技巧。"
"你还知道什么隐藏功能？**评论区**告诉我！"
```

### 模板 2：情景剧 (Skit / 短剧)

**适用平台：** 抖音、TikTok
**推荐时长：** 30-90秒

```
【标题】[角色/场景] + [冲突/反转]
示例：当程序员女友发现男友的代码写得很烂

【场景设定】
- 人物：A（主角），B（配角）
- 场景：[办公室/家/咖啡厅]
- 情绪基调：[搞笑/感动/讽刺]

【开头 — 冲突引入（0-5秒）】
[画面] A 正在做某事
B: "你在干什么？"
A: [一句引发冲突的话]

【发展 — 矛盾升级（5-20秒）】
A: [进一步的行为/台词]
B: [反应，矛盾升级]
[此处可插入第三方角色增加戏剧性]

【高潮 — 反转（20-30秒）】
[出乎意料的转折]
[可以是真相揭露/角色逆转/意外结局]

【结尾 — 点题（最后3秒）】
[一句点题的台词或画面定格]
[配合BGM情绪变化]

【字幕卡】
"关注 @XXX，每天更新"
```

### 模板 3：教程类 (Tutorial / How-to)

**适用平台：** B站、YouTube（长版）；抖音、TikTok（短版）
**推荐时长：** 短版 60秒 / 长版 5-15分钟

```
【标题】"如何[实现某个目标]" / "[数字]步教你[达到效果]"
示例：5分钟学会用 Figma 做一个 App 界面

【开头 — 展示成果（0-5秒）】
"先看最终效果——"
[展示完成品画面 3 秒]
"接下来我一步一步教你做"

【步骤1（含画面演示）】
"**第一步**，[操作名称]"
[屏幕录制/实拍操作过程]
"这里有一个小技巧：[tips]"
"注意，[常见错误提醒]"

【步骤2】
"**第二步**，[操作名称]"
[演示]
"如果你遇到 [常见问题]，那是因为 [原因]，解决方法是 [方案]"

【步骤3】
"**第三步**，也是最关键的一步"
[演示]

【... 根据需要增加步骤】

【成果对比】
"来看看前后对比——"
[Before / After 画面]

【结尾 CTA】
"完整素材/代码/模板我放在评论区/简介栏了"
"如果你学会了，点个赞让我知道！"
"有问题评论区问我，我会回复的"
```

### 模板 4：Vlog (视频日志)

**适用平台：** B站、YouTube（长版）；抖音（短版）
**推荐时长：** 短版 60秒 / 长版 8-15分钟

```
【标题】[时间/地点] + [事件] + [情绪/价值]
示例：在东京生活一个月后，我后悔了

【开头 — 场景带入（0-5秒）】
[环境空镜 + BGM渐入]
旁白/独白："[时间/地点]，[引出主题的一句话]"

【场景1 — 日常/起因】
[手持跟拍，自然光]
"今天我要去 [地点/做某事]"
[展示准备过程/出发画面]

【场景2 — 核心体验】
[多角度拍摄，展示细节]
"[分享感受/见闻]"
[穿插空镜、美食、风景等 B-roll]

【场景3 — 高潮/亮点】
[最精彩的部分]
"[真实反应/情绪表达]"

【场景4 — 反思/感悟】
[安静的场景，可以是夜晚/咖啡厅]
"[总结感悟，升华主题]"

【结尾】
[回扣开头的画面或台词]
"这就是我在 [地点] 的 [时间段]"
[BGM渐出，画面渐黑]
```

### 模板 5：带货种草 (Product Review / 种草)

**适用平台：** 抖音、TikTok（短版）；B站、YouTube（长版）
**推荐时长：** 30-60秒（短版）/ 5-10分钟（长版）

```
【标题】[痛点] + [产品] + [效果]
示例：油皮救星！这款粉底液让我一整天不脱妆

【开头 — 痛点共鸣（0-3秒）】
"[描述一个大家都有的痛点]"
"我之前也是这样，直到我发现了这个——"
[产品入镜]

【产品介绍（3-10秒）】
"这是 [品牌] 的 [产品名]"
"它最大的特点就是 [核心卖点1]"
[近景/特写展示产品]

【使用演示（10-30秒）】
"来看看实际效果"
[上手使用/上脸/安装等实际演示]
"你看，[效果描述]"
[Before/After 对比]

【优势总结（30-45秒）】
"我最喜欢它的三个点："
"第一，[优势1]"
"第二，[优势2]"
"第三，[优势3]"
"当然也说一下不足：[客观缺点，增加可信度]"

【促单 CTA（45-60秒）】
"现在 [平台] 有活动，[优惠信息]"
"链接我放在 [购物车/简介栏/评论区]"
"想要的姐妹/兄弟赶紧冲！"
[强调限时/限量，制造紧迫感]
```

## 四、字幕格式规范 (Subtitle Format Standards)

### SRT 格式 (SubRip Text)

SRT 是最通用的字幕格式，几乎所有剪辑软件都支持导入。

```
序号
开始时间 --> 结束时间
字幕文本（最多两行）

```

**完整示例：**

```srt
1
00:00:00,000 --> 00:00:02,500
你知道吗？

2
00:00:02,500 --> 00:00:05,000
90%的人都在犯
这个错误

3
00:00:05,000 --> 00:00:08,500
今天我来告诉你
正确的做法

4
00:00:08,500 --> 00:00:12,000
第一步
打开你的手机设置

5
00:00:12,000 --> 00:00:15,500
找到"电池"选项
点进去
```

### SRT 规范要点

| 规则 | 中文 | English |
|------|------|---------|
| 每行最大字符数 | 18个汉字 | 36 characters |
| 每屏最多行数 | 2行 | 2 lines |
| 单条最短时长 | 1秒 | 1 second |
| 单条最长时长 | 5秒 | 5 seconds |
| 时间码格式 | HH:MM:SS,mmm | HH:MM:SS,mmm |
| 条目间隔 | 空一行 | One blank line |
| 断句原则 | 在语义完整处断开 | Break at natural phrase boundaries |
| 标点处理 | 省略句号，保留？！ | Omit periods, keep ? ! |

### 双语字幕示例

```srt
1
00:00:00,000 --> 00:00:03,000
你知道吗？90%的人都在犯这个错误
Did you know? 90% of people make this mistake

2
00:00:03,000 --> 00:00:06,500
今天我来教你正确的方法
Today I'll show you the right way
```

## 五、各平台最佳实践 (Platform Best Practices)

### 抖音 (Douyin)

| 维度 | 规格 |
|------|------|
| 画面比例 | 9:16 竖屏 (1080x1920) |
| 推荐时长 | 15s / 30s / 60s / 2-3min |
| 最佳发布时间 | 工作日 12:00-13:00, 18:00-20:00; 周末 10:00-11:00 |
| 封面规格 | 1080x1920, 文字居中上半部分 |
| 标题字数 | 15-30字，含关键词和 hashtag |
| 字幕位置 | 画面中下部，避开底部互动栏 |
| 内容调性 | 节奏快、信息密度高、情绪强烈 |
| BGM | 优先使用抖音热门音乐，增加推荐权重 |

### B站 (Bilibili)

| 维度 | 规格 |
|------|------|
| 画面比例 | 16:9 横屏 (1920x1080) |
| 推荐时长 | 5-15分钟 |
| 最佳发布时间 | 工作日 17:00-19:00; 周末 10:00-12:00 |
| 封面规格 | 1920x1080 或 1146x717, 文字醒目 |
| 标题字数 | 20-40字，可用【】标注分类 |
| 字幕 | 支持 CC 字幕上传，建议 SRT 格式 |
| 内容调性 | 深度、有梗、人格化、重视弹幕互动 |
| 分P | 长内容可分P，每P有独立标题 |

### YouTube

| Item | Spec |
|------|------|
| Aspect Ratio | 16:9 landscape (1920x1080 or 4K) |
| Recommended Length | 8-15 minutes (for ad monetization: 8min+) |
| Best Upload Time | Tue-Thu, 2-4 PM local time of target audience |
| Thumbnail | 1280x720, high contrast, large text, face close-up |
| Title | 60-70 chars, front-load keywords |
| Subtitle | Upload SRT via YouTube Studio; auto-captions available |
| Content Style | High production value, structured, SEO-optimized |
| End Screen | Last 20s: subscribe button + recommended video |

### TikTok

| Item | Spec |
|------|------|
| Aspect Ratio | 9:16 vertical (1080x1920) |
| Recommended Length | 15-60 seconds |
| Best Upload Time | Mon-Fri 7-9 AM, 12-3 PM, 7-11 PM (target timezone) |
| Cover | First frame or custom thumbnail |
| Caption | 150 chars max, include 3-5 hashtags |
| Subtitle Position | Center or lower-center, avoid bottom bar |
| Content Style | Trend-driven, music-forward, fast-paced |
| Sound | Use trending sounds for algorithm boost |
